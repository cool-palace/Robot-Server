<h1>Сервер для управления 6-осевым роботом</h1>

Данная программа представляет собой сервер, симулирующий управление 6-осевым роботом по протоколу TCP и предоставляюший координаты манипулятора по командам клиентов. Для запуска сервера в качестве аргумента необходимо передать номер порта, например:

<code>server.exe 3000</code>

Используемая математическая модель робота рассмотрена в программе: [Математическая модель 6-осевого робота](https://github.com/cool-palace/Collaborative-Robot). Эту же программу можно использовать для верификации результатов расчёта.

<h2>Возможности сервера</h2>

На данный момент сервер осуществляет обработку запросов на расчёт текущих координат робота и на поворот его отдельных сочленений, после чего также выдаёт итоговые координаты.  

Для работы с данным сервером предназначен клиент: [Клиент для управления 6-осевым роботом](https://github.com/cool-palace/Robot-Client). Сервер поддерживает подключение нескольких клиентов и обработку из запросов в отдельных потоках.

Поворот робота симулируется сервером с заданной конечной скоростью, конечные координаты выдаются по окончании операции поворота. Если запрос на поворот робота приходит в момент, когда робот уже находится в процессе движения, то запрос ожидает своей очереди для выполнения.
